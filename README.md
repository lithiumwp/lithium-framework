# Lithium Framework

A modern PHP framework for building WordPress plugins.


## Report an Issue

Feel free to [open an issue](https://github.com/lithiumwp/lithium-framework/issues/new) if one does not [already exist](https://github.com/lithiumwp/lithium-framework/issues).


## Code of Conduct

Please review and adhere to our [Code of Conduct](https://github.com/lithiumwp/.github/blob/main/CODE_OF_CONDUCT.md) to ensure a welcoming and respectful environment for all contributors.


## Security Vulnerabilities

Please review our [Security Policy](https://github.com/lithiumwp/lithium-framework/security/policy) for information on how to report security vulnerabilities.
